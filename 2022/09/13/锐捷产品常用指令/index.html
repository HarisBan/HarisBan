一、锐捷产品常用指令


1.命令模式及切换

| 模式 | 切换 | 提示符 | 简要描述 |
|:-: |:—- |:——|:——–: |
|用户
模式|enable -> 特权模式(可能需要密码）
exit -> 离开|Ruijie>|用来进行基本测试、显示系统信息|
|特权
模式|disable -> 返回用户模式
configure -> 全局配置模式|Ruijie#|验证结果、具有口令保护（?保护啥,不理解）|
|全局
配置|exit、end、Ctrl+C -> 返回特权模式
interface -> 进入接口配置模式（需要指明具体哪一个接口)
vlan vlan_id -> VLAN配置模式 |Ruijie(config)#|配置全局参数|
|接口
配置|end、Ctrl+C -> 特权模式
exit -> 返回全局配置模式|Ruijie(config-if)#| 配置具体的接口|
|VLAN
配置|end、Ctrl+C -> 特权模式
exit -> 返回全局配置模式|Ruijie(config-vlan)#|配置Vlan的参数|
